# 보안 민감 영역

- `.env`·비밀키 값을 **출력·로그·커밋 금지**. 하드코딩 금지(환경변수/시크릿 매니저 사용).
- SQL은 **반드시 parameterized query** — 문자열 보간 금지.
- 인증 흐름 변경, 권한 로직 수정은 **사람 승인 후** 진행.
- DB 마이그레이션·스키마 변경은 dry-run/리뷰 후 실행. 되돌릴 수 있게 작성.
