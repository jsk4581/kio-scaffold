# .dev/ — AI가 남기는 작업 기록 (작업의 흔적)

AI가 작업 **중·후에** 배운 것을 남기는 곳. 사람이 관리하는 `docs/`와 섞지 않는다.

권장 사용:
- `learnings/` — 막혔던 지점과 해결법(다음 세션이 같은 실수를 반복하지 않게)
- `troubleshooting.md` — 디버깅 히스토리
- `worklog/` — 세션별 작업 로그
- `scratch/` — 실험·임시 산출물
- `collab/` — 에이전트 간 협업: 인수인계서·작업 보드·작업 선점 (여러 에이전트/세션이 이어서·동시에 작업할 때만). 규약: `collab/README.md`

> 여기 쌓인 패턴/실수가 반복되면 → `docs/` 또는 `.claude/rules/`로 승격시킨다(개선 루프).
