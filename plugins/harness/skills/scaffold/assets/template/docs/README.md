# docs/ — 사람이 관리하는 문서 (비즈니스의 진실)

이 폴더는 **사람의 책임**이다. AI는 작업 **전에** 여기를 참고한다.
한꺼번에 다 읽히지 말고, `CLAUDE.md`/규칙에서 "이 상황엔 이 문서를 봐"로 **상황별 참조**시킨다.

권장 파일:
- `architecture.md` — 시스템 구조, 레이어, 의존성 방향
- `conventions.md` — 코딩/네이밍/커밋 컨벤션
- `glossary.md` — 도메인 용어 정의
- `adr/` — 주요 의사결정 기록(Architecture Decision Records)
- `api/` — API 스펙, 외부 연동 규격

> 비교: 사람이 남기는 진실은 `docs/`, AI가 남기는 작업 흔적은 `.dev/`.
